System Design

Layered Architecture

Layer 1 · Presentation
Frontend Layer
Next.js (React) · Tailwind CSS
Renders UI, handles user input, calls backend via API routes, manages local state. Screens: Wardrobe, Upload, Stylist, Matcher, Shop.
Layer 2 · Business Logic
API Routes Layer
Next.js API Routes
Four independent modules: Digital Closet, AI Style Matcher, Intelligent Concept Stylist, Commerce Integration. Factory Method design pattern applied here.
Layer 3 · AI & External Services
Intelligence Layer
Google Gemini Vision API · Google Custom Search API
Zero-shot clothing classification, style descriptor generation, real product search. Results cached to stay within free tiers.
Layer 4 · Data & Infrastructure
Data Layer
Supabase (PostgreSQL, Auth, Storage) · Railway.app
Persists user data, clothing metadata, images. Provides JWT-based authentication. Docker containerised, cloud deployed.

Continuous Deployment

Automated Pipeline

Step 01
Push to main
Developer merges a pull request or pushes directly to the main branch on GitHub.
Step 02
CI Triggered
GitHub Actions detects the push and starts the build-and-test job automatically.
Step 03
Tests Run
Flake8 lint check runs first, then unittest + coverage report. Must be ≥ 70% to pass.
Step 04
Deploy Job
Only if tests pass, the deploy job fires. It calls the Render deploy hook via curl.
Step 05
Live 🎉
Render pulls the repo, builds the Docker image (nginx:alpine), and serves it live.
